# CodeAlpha Car Price Prediction

## Objective
Build a machine learning model to predict car prices using regression.

## Tasks Performed
- Data preprocessing
- Feature engineering
- Model training
- Model evaluation
- Visualization of predictions

## Tools Used
Python, Pandas, Scikit-learn, Matplotlib, Streamlit

## Run Project

Install dependencies

pip install -r requirements.txt

Run Streamlit

streamlit run streamlit_app.py
