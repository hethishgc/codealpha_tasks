# CodeAlpha Task 3: Car Price Prediction using Machine Learning

## Objective
Develop a machine learning model to predict car selling prices based on vehicle features.

## Dataset
The dataset includes information about used cars such as:

- Car Name
- Year
- Present Price
- Kilometers Driven
- Fuel Type
- Selling Type
- Transmission
- Owner
- Selling Price (target variable)

## Tasks Performed
- Data preprocessing and cleaning
- Feature engineering using categorical encoding
- Train-test split of dataset
- Regression model training using Linear Regression
- Model evaluation using MAE and MSE
- Visualization of actual vs predicted prices

## Tools Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit

## Model
A Linear Regression model was trained to estimate the selling price of a car based on its attributes.

## How to Run

Install dependencies:

pip install -r requirements.txt

Run the Streamlit dashboard:

streamlit run streamlit_app.py

## Outcome
The project demonstrates how machine learning can be applied to predict car prices using historical vehicle data and regression techniques.
