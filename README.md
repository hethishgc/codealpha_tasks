# CodeAlpha Data Science Internship Projects

This repository contains the projects completed during the **CodeAlpha Data Science Internship**.
The projects demonstrate practical applications of **machine learning, data analysis, and visualization using Python**.

---

## Projects Included

### 1. Iris Flower Classification

**Objective:**
Build a machine learning model to classify iris flowers into three species:

* Setosa
* Versicolor
* Virginica

**Approach:**

* Loaded Iris dataset
* Performed exploratory data analysis
* Trained a classification model using Scikit-learn
* Evaluated model predictions

**Technologies Used**

* Python
* Pandas
* Scikit-learn
* Matplotlib
* Seaborn

---

### 2. Unemployment Analysis with Python

**Objective:**
Analyze unemployment trends and understand the **impact of COVID-19 on unemployment rates in India**.

**Approach:**

* Data cleaning and preprocessing
* Visualization of unemployment trends
* Region-wise unemployment analysis
* COVID-19 impact comparison
* Identification of seasonal patterns

**Technologies Used**

* Python
* Pandas
* Matplotlib
* Seaborn

---

### 3. Car Price Prediction using Machine Learning

**Objective:**
Develop a machine learning model to **predict car prices based on vehicle features**.

**Features Used**

* Year
* Present Price
* Kilometers Driven
* Fuel Type
* Transmission
* Ownership history

**Approach:**

* Data preprocessing and feature engineering
* Training a regression model
* Model evaluation using MAE and MSE
* Visualization of actual vs predicted prices

**Technologies Used**

* Python
* Pandas
* Scikit-learn
* Matplotlib
